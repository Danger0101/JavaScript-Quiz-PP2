How to play

  1. Click Play and wait for quiz to load
  2. Choose the answer you think is correct once quiz has loaded
  3. Upon choosing the answer will highlight incorrect (red) and correct (green) for exactly one second then it will load next question
  4. The progress bar and question counter will also update if answer was correct then score will increase
  5. Once last question is complete the quiz will end
  6. If you wish to save your score, input your username and hit save this will return you to the menu and save your score
  • If you wish to retun to menu at any point hit the menu or home buttons
  • After quiz there is a button to play game again and re launches the quiz
  • You can view the leaderboard at any point through the main menu
